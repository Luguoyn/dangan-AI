# 与弹丸论破原版游戏的差异

> 对比对象：《弹丸论破1·希望的学园与绝望的高中生》《超级弹丸论破2·再见绝望学园》《新弹丸论破V3·大家的自相残杀新学期》  
> 当前版本：v0.15  
> 引擎：Godot 4.x  
> 更新日期：2026-05-05

## 一、已实现功能总览

### 1.1 日常篇（2D）

| 功能 | 状态 | 说明 |

| 2D 场景渲染 | ✅ | 网格地板 + 暗色背景 |

| 玩家视觉 | ✅ | 绿色矩形 + "苗木" 标签 |

## 三、与原版设计理念的本质差异

### 3.1 架构差异

| 维度 | 原版 | 本作 |
|------|------|------|
| 引擎 | 自研/Unity | Godot 4.x |
| 剧本系统 | 硬编码 + 脚本 | JSON 驱动（27 指令处理器） |
| 美术资源 | 手绘立绘 + 3D 模型 | 纯色几何体（CSG + ColorRect） |
| 音频 | 全程配音 + OST | AudioManager 就绪，资源待填 |
| 可扩展性 | 需改源代码 | 改 JSON 剧本即可扩展章节 |

### 3.2 无休止议论差异

| 维度 | 原版 | 本作 |
|------|------|------|
| 矛盾标记 | 整句黄色高亮 | 句中词 hotspot（真假区分） |
| 言弹选择 | 光标在文字上时自动高亮匹配言弹 | Tab 切换，需手动匹配 |
| 言弹发射 | 蓄力 + 子弹飞行动画 | 即时发射 + 文字从右飞入 |
| 杂音干扰 | 白噪音文字覆盖屏幕 | 半透明灰色文字漂浮 |
| 发言循环 | 固定轮数 | 无限循环直到击破或 HP 归零 |
| 错误反馈 | 简短语音"違う！" | 可配置底部对话框 |

### 3.3 摄像机差异

| 维度 | 原版 | 本作 |
|------|------|------|
| 运镜风格 | 快速 cut + 特写 shake | 平滑旋转 + 锁定微漂 |
| 配置方式 | 硬编码 | JSON 可配置 4+4 预设 |
| 3D 场景 | 预烘焙 + 后处理 | 实时 CSG 几何体 |

### 3.4 整体内容量差异

| 维度 | 原版单个作品 | 本作 |
|------|------------|------|
| 完整章节 | 6 章 + 序章 | 0（框架完成，无剧情内容） |
| 剧本长度 | 约 30 万字 | 约 500 字（演示剧本） |
| 角色数量 | 16 名有完整剧情 | 17 名数据就绪，3 名有演示对话 |
| 学级裁判场次 | 6 场 | 0（仅测试模式） |
| 游戏时长 | 25-35 小时 | 约 5 分钟（演示内容） |

## 五、后续开发建议（按优先级）

### 高优先级（完善核心体验）
1. 编写完整的第一章日常→非日常→学级裁判剧本
2. 为所有 17 个角色编写演示对话
3. 实现尸体发现 + 黑白熊档案
4. 实现处刑演出场景
5. 完善拼字的倒计时和难度曲线

7. 实现机关枪论战 / 逻辑深潜
8. 实现群体辩论（Scrum Debate）
