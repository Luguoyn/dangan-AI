### v0.15 — 无休止议论深度改造 (2026-05-05)

**30+ 次提交，涵盖议论系统重构、摄像机运镜配置、言弹交互、BUG修复**

#### 议论核心机制改造

| 特性 | 说明 |
|------|------|
| 句中矛盾词高亮 | `hotspot` 模式取代整句矛盾，一句话可含多个真/假矛盾词 |
| 轮流发言循环 | `speak_duration` 控制每条发言时长，轮完未击破→主角思考→重新循环 |
| 主角总结思考 | 裁判场对话框复用，隐藏议论遮罩后显示 |
| 唯一真矛盾 | 多个假矛盾（橙色）+ 1 个真矛盾（金色），只有真矛盾+正确言弹才能击破 |
| 可配置错误对话 | 每个 hotspot 可配置 `fail_dialogue`，击错后显示并重启议论 |
| BREAK 特效 | 白闪 + 红色大字 + 12 片彩屑飞出 + 闪烁动画（1.5s） |

#### 言弹系统演变

| 版本 | 交互方式 |
|------|---------|
| 初版 | 环形菜单：Tab 打开，鼠标点击选择 |
| 第二版 | Tab 循环切换，无列表 |
| 最终版 | **点按 Tab**（<0.3s）→ 切换言弹；**长按 Tab** → 弹出列表 + 滚轮选择 + 松开确认 |

#### 摄像机运镜系统

| 类别 | 预设 | 说明 |
|------|------|------|
| 发言切换（`camera_transition`） | `rotate` | 绕中心旋转（默认） |
| | `cut` | 瞬间切换 |
| | `sweep` | 先拉远 wide 再切近 |
| | `zoom` | 退远后快速推进 |
| 锁定后（`camera_lock`） | `drift` | 微漂移呼吸感（默认） |
| | `steady` | 完全静止 |
| | `breathe` | 缩放呼吸 |
| | `dolly` | 横向滑轨 |

配置方式：`debate_test_01.json` 中 `camera_transition`/`camera_lock` 字段。

#### 言论射击交互

| 特性 | 说明 |
|------|------|
| 言弹动画 | 证据名文字从屏幕右侧 2x 缩放飞入（3D 纵深感） |

| 失败不换言弹 | 仅成功击破后清空，失败保持当前选中 |
