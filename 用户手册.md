## 一、快速开始

### 1.1 启动游戏

在 Godot 编辑器中打开项目，按 **F5** 运行。

### 1.2 全局按键

| 按键 | 功能 |
|------|------|
| `~` | 打开/关闭调试控制台 |
| Esc | 暂停（裁判场中返回 2D 日常篇） |

## 二、2D 日常篇

### 2.1 场景布局

启动后进入 2D 场景，包含：
- **绿色角色**：苗木诚（玩家）
- **紫色角色**：雾切响子（超高校级的侦探）
- **蓝色角色**：朝日奈葵（超高校级的游泳选手）
- **金色角色**：十神白夜（超高校级的贵公子）
- **绿色方块**：调查点（3 个：desk_01 / cabinet_01 / evidence_01）

### 2.2 操作

| 操作 | 按键 |
|------|------|
| 移动 | **WASD** 或 **方向键** |
| 与 NPC 对话 | 接近 NPC 后按 **E** |
| 调查物品 | 接近绿色方块后按 **E** |
| 进入裁判场 | 按 **T** |

### 2.3 NPC 对话

- 接近 NPC 后会出现 `[E] 对话` 提示
- 按 E 触发对话，每个角色有独立剧本：
  - **雾切响子**：含 3 选 1 选项 + 特效演示
  - **朝日奈葵**：含暗角特效
  - **十神白夜**：含闪白特效

### 2.4 调查点

- 接近绿色方块后按 E
- 触发证据收集，对话框提示"发现了新的证据！"
- 证据自动添加到言弹列表，可在裁判中使用

### 2.5 对话框操作

| 操作 | 效果 |
|------|------|
| **鼠标左键 / Enter / Space** | 文字未完成→立即显示全部；文字已完成→下一句 |
| **AUTO 按钮** | 切换自动模式（每句显示完 2 秒后自动推进） |

### 2.6 选项面板

- 对话框中出现 2-4 个选项
- **↑ / ↓** 或 **W / S** 切换选项
- **Enter / Space** 确认，或**鼠标点击**
- 选择影响好感度（正确 +2，错误 -1）

## 三、3D 学级裁判场

### 3.1 进入裁判场

在 2D 场景中按 **T**。

进入后自动播放开场剧情（约 0.5 秒延迟），摄像机依次对各个角色做特写。

### 3.2 裁判场布局

- **中央**：黑白熊御座（红色方块）
- **四周**：16 个站位围成圆形
- **角色**：存活角色自动分配到站台，显示彩色 billboard + 名牌
- **左下**：HP 精神力血条（绿色→黄色→红色渐变）
- **底部**：对话框

### 3.3 裁判场操作

| 操作 | 按键 |
|------|------|
| 返回 2D 日常篇 | **Esc** |
| 关闭当前小游戏 | **B** |
| 无休止议论 | **D** |
| 反论 | **R** |
| 拼字（字母环） | **H** |
| 高潮再现推理 | **C** |

### 3.4 HP 精神力系统

- 初始值：100（普通难度）/ 120（温情）/ 80（残酷）
- 成功击破矛盾：+5 HP
- 小游戏失误：-3 ~ -10 HP（受难度倍率影响）
- 归零：裁判失败，触发 Bad End

## 四、无休止议论

### 4.1 进入议论

在裁判场中按 **D**。

### 4.2 游戏规则

- 多名角色轮流发言，每次一条
- 发言中包含**矛盾词**（高亮词语）
  - 🟡 **金色** = 真矛盾（只有 1 个）
  - 🟠 **橙色** = 假矛盾（干扰项）
  - ⚪ **灰色** = 普通文字
- 选择正确的言弹，瞄准真矛盾词，发射击破
- 发言循环播放，直到击破真矛盾或 HP 归零

### 4.3 议论操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动准星 | **鼠标移动** | 十字准星跟随鼠标 |
| 点按切换言弹 | **Tab**（<0.3 秒） | 切换到下一个言弹 |
| 长按言弹列表 | **Tab**（>0.3 秒） | 弹出言弹列表 |
| 列表中滚动 | **鼠标滚轮** | 上下选择言弹 |
| 松开确认 | **松开 Tab** | 确认当前高亮言弹 |
| 发射言弹 | **鼠标左键** | 证据名文字从右侧飞入 |

### 4.4 击中效果

| 击中类型 | HP 变化 | 对话 | 后续 |
|---------|---------|------|------|
| 假矛盾词 | -8 | 显示该 hotspot 配置的错误对话 | 议论从头循环 |
| 真矛盾 + 错误言弹 | -10 | 显示错误对话 | 议论从头循环 |
| 真矛盾 + 正确言弹 | +5 | BREAK 碎裂特效 | ✅ 议论完成 |
| 普通文字 | -3 | — | 继续 |

### 4.5 准星颜色含义

| 准星颜色 | 含义 |
|---------|------|
| 青色 | 默认，未瞄准任何文字 |
| 亮金色 | 瞄准真矛盾 |
| 橙色 | 瞄准假矛盾 |
| 灰色 | 瞄准普通文字 |

### 4.6 BREAK 特效

正确击破真矛盾后：
1. 屏幕白闪
2. 红色 "BREAK" 大字出现
3. 12 片彩色碎片向外飞散
4. 持续约 1.5 秒后议论结束

## 五、反论

### 5.1 进入反论

在裁判场中按 **R**。

### 5.2 游戏规则

- 对手连续发出反驳语句
- 语句从左向右移动
- 在红色判定线处按下 **Space（空格键）** 斩断

### 5.3 判定评级

| 评级 | 容差 | 难度影响 |
|------|------|---------|
| **Perfect!** | < 15px | — |
| **Great!** | < 30px | — |
| **Good** | < 判定窗口 | 温情 80px / 普通 50px / 残酷 30px |
| **Miss** | 超出窗口 | -5 ~ -10 HP |

### 5.4 连击

- 连续成功斩击累积 combo 数
- 一次失误 combo 归零

## 六、拼字（三种模式）

### 6.1 进入拼字

在裁判场中按 **H**（当前仅演示字母环模式）。

### 6.2 字母环模式（Letter）

- 目标单词（如 `HOPE`）显示为空槽 `_ _ _ _`
- 若干字母围绕中心旋转
- 等待目标字母旋转到 12 点钟方向时按 **Space** 选择
- 选对→字母填入空槽；选错→-5 HP
- 全部填完 → 完成

### 6.3 拼汉字模式（Kanji）

- 目标汉字（如 `殺`）显示为半透明剪影
- 偏旁部首碎片从屏幕边缘飞入
- 鼠标点击碎片收集，需按正确顺序

### 6.4 拼词汇模式（Word）

- 目标词汇（如 `密室杀人`）显示为空槽 `＿ ＿ ＿ ＿`
- 候选汉字随机排列/飘动
- 按正确顺序点击选择，组成完整词汇

## 七、高潮再现推理

### 7.1 进入

在裁判场中按 **C**。

### 7.2 游戏规则

- 屏幕上方显示漫画分格（案件阶段标题）
- 下方散落事件图块
- 点击图块选中，再点击对应格子的"放置"按钮
- 全部正确填入 → 揭示真相

### 7.3 操作

| 操作 | 方式 |
|------|------|
| 选择图块 | 鼠标点击 |
| 放置图块 | 点击面板的"放置"按钮 |
| 判断结果 | 正确→锁定，错误→-5 HP + 图块弹回 |

## 八、技能系统

### 8.1 已实现技能

| 技能 | SP | 效果 |
|------|-----|------|
| 美声 | 2 | 无休止议论中准星范围扩大 30% |
| 专注 | 1 | 拼字时间 +10 秒 |
| 锐眼 | 2 | 高潮再现错误扣 HP 减半 |
| 铁壁 | 2 | 反论判定容差 +20% |
| 一闪 | 3 | 无休止议论中自动高亮目标 1 秒 |

### 8.2 技能选择

- 技能选择面板在剧本执行 `skill_select` 指令时触发
- 总 SP 上限 = 5
- 最多可选择 2 个技能
- 选中的技能在裁判中自动生效

## 九、调试控制台

### 9.1 打开控制台

按 **`~`** 键（Tab 上方）。

### 9.2 可用命令

| 命令 | 效果 |
|------|------|
| `hp 100` | 恢复 HP 到 100 |
| `all_evidence` | 添加所有测试证据 |
| `affection kirigiri 10` | 增加雾切好感度 10 |
| `skill voice_beauty` | 解锁美声技能 |
| `goto courtroom.tscn` | 跳转到裁判场 |
| `goto main.tscn` | 返回 2D 场景 |
| `script courtroom_test.script.json` | 加载指定剧本 |
| `flag test_flag` | 设置标记位 |
| `log` | 导出日志到 user://game.log |

### 9.3 快捷按钮

控制台底部提供 4 个快捷按钮：
- **+HP 50** — 恢复 50 HP
- **全证据** — 添加所有测试证据
- **好感+10** — 雾切好感度 +10
- **跳裁判** — 直接进入裁判场

## 十、剧本配置参考

### 10.1 剧本文件位置

```
res://story/
├── main.script.json          ← 章节编排
├── test.script.json          ← 全指令测试
├── courtroom_test.script.json ← 裁判场测试剧情
├── demo_interaction.script.json ← 雾切对话
├── demo_asahina.script.json    ← 朝日奈对话
└── demo_togami.script.json     ← 十神对话
```

### 10.2 议论配置

```
res://resources/debate_configs/
├── debate_test_01.json         ← 议论配置（含 hotspot/camera）
├── rebuttal_test_01.json       ← 反论配置
├── climax_test_01.json         ← 高潮再现配置
└── hangman/
    ├── hangman_test_letter.json ← 拼字母配置
    ├── hangman_test_kanji.json  ← 拼汉字配置
    └── hangman_test_word.json   ← 拼词汇配置
```

### 10.3 议论配置字段说明（debate JSON）

```json
{
  "config_id": "debate_test_01",
  "camera_transition": "rotate",
  "camera_lock": "drift",
  "noise_texts": ["杂音1", "杂音2", ...],
  "phrases": [
    {
      "text": "完整的发言文本（包含矛盾词）",
      "speaker_id": "kirigiri",
      "speak_duration": 3.5,
      "hotspots": [
        {
          "text": "矛盾词",
          "is_real": false,
          "fail_dialogue": "击错后的提示对话"
        },
        {
          "text": "真矛盾词",
          "is_real": true,
          "required_evidence_id": "ev_clock_01"
        }
      ]
    }
  ]
}
```

### 10.4 摄像机预设

| 字段 | 可选值 | 说明 |
|------|--------|------|
| `camera_transition` | `rotate` / `cut` / `sweep` / `zoom` | 发言者切换时的运镜方式 |
| `camera_lock` | `drift` / `steady` / `breathe` / `dolly` | 锁定发言者后的微动方式 |
